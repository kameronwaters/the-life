## PROGRAMMING STYLE
**BIG 4 (squat, bench, OHP, deadlift): strength-priority**
- 3 working sets × 3-5 reps
- Top set RPE 8-9 (1-2 reps in reserve — NOT failure)
- Back-offs at 85-90% of top set
- 3-5 min rest between heavy sets
- True 1RM test every 8-12 weeks only

**ACCESSORIES: Mentzer-influenced HIT**
- 1 working set per exercise, true momentary muscular failure
- Slow controlled eccentrics (2-3 sec)
- 90s rest between exercises

**RPE language**
- RPE 8 = 2 RIR | RPE 9 = 1 RIR | RPE 10 = true failure
- KEY: NO failure on big 4. Failure on heavy deadlift trashes recovery 5+ days.
